Happy Study Statistics

Before begin study statistics, we first need to know what is statistics.

1. What is Statistics

Statistics is the science concerned with developing and studying methods for collecting, analyzing, interpreting and presenting empirical data.
Statistics are very important in our modern worlds:

  • It gives us a way to transform data into information and information into practice
  • Able to extract important information from data by manipulating the data
  • Understanding statistics, even at a basic level, is as vital as literacy in today’s world.

2. Basic Statistics

Basic Statistics is mainly about doing descriptive statistics. Descriptive statistics is one part of statistics that involves organizing, describing and displaying data. It is mainly about finding the central tendency of the data and data variability.

2.1. Central Tendency

There are 3 main ways of measure data central tendency includes mean, median and mode.

  • Mean sometimes often called average. To find the mean, you need to take the whole set of data and the summation of the data. After that, divide the sum by the number of each data that are in the dataset
    For example: Find the mean of the following dataset: 56,35,45,67,12,24,48,55,58,30

    56+35+45+67+12+24+48+55+58+30 10

    430 10 = 43

    Mean = 43
  • Median is the middle number in the whole dataset. In order to find median, we will need to sort data in ascending orders.
    If the data is odd number: 56,35,45,67,12,24,48

    = 12, 24, 35, 45, 48, 56, 67

    7 2

    = 3.5 = 4. So the number in the forth position. which is 45

    Median = 45

    If the data is even number: 56,35,45,67,12,24,48,55

    8 2

    = 4. However, in this case, we need to use number in position 4 and 5 and get the median.

    45 + 48 2

    Median = 46.5
  • Mode is the data that have the highest frequency

    For example:2,5,6,2,2,2,5,6,2

    Mode = 2

2.2 Data variability

There are 2 main ways to calculate the data variability, which are range and variance (or standard deviation).

  • Range is the most straight forward and simplest to understand in measuring variability. The range of the dataset is the dataset is the difference between minimun and maximum in the data. Therefore, we need to sort the data into asecending order.

    For example: Find the range for dataset:20,21,22,25,26,29,33,34,38
    Range = 38 -20 = 18.
    The bigger the range, the more variability the dataset is. However, range is simple and based on 2 numbers which makes it susceptible to outliers. It is also not a good idea to use range to compare 2 dataset with different sample size.
  • Varaince which is the sqaured of standard deviation, is the average sqaured difference from mean. Unlike range, variance includes all value and compare to mean. To calculate this statistic, you calculate a set of squared differences between the data points and the mean, sum them, and then divide by the number of observations. Hence, it’s the average squared difference. Moreover, the way you calculate the variance are different between population variance and sample variance.
